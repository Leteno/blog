我们会以下几个方面讨论 C++ 跟 Java 的区别：

* C++ 对象的[内存管理](MemoryManagement.md)
* 复制？引用？C++ 的[参数传递](ParameterPassing.md)
* [析构及其其他默认函数](DefaultFunction.md)
